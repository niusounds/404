# 呪われたデスクトップ：怪異仕様書

このドキュメントでは、デスクトップ環境で発生する怪異（ホラー演出）の発生条件、仕組み、および各フェーズの変遷について解説します。

---

## 核心システム：侵食度 (corruptionLevel)
すべての怪異は内部変数 `corruptionLevel` によって制御されます。この値が上昇するにつれて、PCの挙動は致命的に崩壊していきます。

### 侵食度が上昇するアクション
1.  **ファイルの閲覧**: `README.txt` を開く、または各種フォルダ内のファイルを開く。
2.  **システムへの接触**: 「ユーザーアカウント」画面を開く、または「Run (実行)」コマンドを使用する。
3.  **警告の無視**: 表示される警告ウィンドウを閉じる。
4.  **タブの切り替え**: 別タブへ移動し、しばらくしてから戻る（4回目以降）。
5.  **死の淵**: BSOD（ブルースクリーン）演出が終了した際。

---

## 侵食段階別の演出 (Phase Progression)

### 第1段階：違和感 (Level 1~2)
*   **READMEの忠告**: READMEを開くことで物語が始動する。

### 第2段階：胎動 (Level 3~4)
*   **ウィンドウの震動**: 各ウィンドウが小刻みに震え始める。
*   **背景の予兆**: 画面が暗くなり、背景にうっすらと「顔」のシルエットが浮かび上がる。
*   **囁き**: 「みてるよ」という低い囁き声。
*   **焼き付き**: ウィンドウをドラッグした際、タイトルバーの残像が画面に残るようになる。

### 第3段階：汚染 (Level 5~6)
*   **タイピング・ハイジャック**: メモ帳で文字が打てなくなり、代わりに「たすけて」等の不気味な文章が勝手に打ち込まれる。
*   **ラベルの変異**: アイコンの名前が「しね」「404」「遺体」などに書き換わる。
*   **反応する震動**: 何かをクリックするたび、画面全体が激しく揺れ、グリッチノイズが走る。

### 第4段階：高濃度侵食 (Level 6~9)
*   **監視眼の出現**: タスクバーに「目」が現れ、マウスカーソルを執拗に追いかけ始める。
*   **空間の歪み**: 画面全体が3D的に歪み、色収差（色彩のズレ）が発生する。
*   **背後の写り込み**: 白いウィンドウの背景に「何者か」が反射して見えるようになる。
*   **自律行動**: 誰もいないのにアイコンが勝手に選択されたり、スタートボタンの文字が「SAVE ME」等に変異する。
*   **エンティティ・チャット**: 見知らぬ誰かとチャットが繋がり、あなたの部屋の状況（後ろの扉など）を指摘される。

### 第5段階：崩壊 (Level 10)
*   **BSOD (Blue Screen of Death)**: 致命的なエラーとしてブルースクリーンが発生。エラーコードには「BEHIND_YOU（お前の後ろ）」といったメッセージが含まれる。

### 第6段階：悪夢 (Level 11+)
*   **血の滲出**: 全ウィンドウの縁から赤い液体が滲み出し、滴り落ちる。
*   **重力異常**: 画面が突如として上下反転する。
*   **マウスの呪い**: カーソルの形が変わり、ウィンドウがあなたの手を逃れるように勝手に動く。
*   **スパムと悲鳴**: 大量の警告ウィンドウとジャンプスケア（叫び声や画像）がランダムに発生。

### 第7段階：終末 (Level 13+)
*   **死のカウントダウン**: 時計が時刻を捨て、0秒へ向けてカウントを開始する。0になった瞬間、最大の恐怖が訪れる。

## 超低確率・特殊ギミック (Rare Gimmicks)

1.  **一瞬の残像 (0.5%)**: 
    侵食度が上昇する際、0.05秒だけ全画面に死に顔が映り込む。
2.  **重力崩壊 (0.2%)**: 
    画面内のアイコンが突然重力を失い、画面下部にバラバラと崩れ落ちる。
3.  **プログラムの反乱 (5%)**: 
    スタートメニューの「Programs」を開こうとすると、稀にエラーウィンドウが20枚以上一気に開く。
4.  **静寂の囁き (無操作30秒後)**: 
    放置されている間、PCの背後から「なぜ、そこにいるの？」とクリアな囁きが聞こえる。

---

## 終了演出：シャットダウン
侵食が進んだ状態で「Shut Down」を選択した場合、以下の演出が発生します。
1.  **どこへいくの？**: 低い声での拒絶。
2.  **溶解 (Melt down)**: 画面全体が熱で溶け落ちるように崩壊。
3.  **完全停止**: 「もう、おそい」というメッセージと共に画面が黒く染まり、すべての操作が不能になる。

## 技術的実装の詳細 (Technical Implementation Details)

このプロジェクトでは、標準的なWeb技術をクリエイティブ（かつ不気味）な方法で組み合わせて使用しています。

### 1. 視覚効果 (Visual Effects)
*   **CSS 3D Transforms & Perspectives**: 
    `perspective` と `rotateX/Y` を組み合わせることで、平面であるはずのブラウザ画面が3D空間で歪んでいるような効果（Screen Warp）を実現しています。
*   **CSS Filters**: 
    侵食度に応じて `blur`, `contrast`, `grayscale`, `invert` などのフィルタを動的に適用し、視覚的な崩壊を表現しています。
*   **Chromatic Aberration (色収差)**: 
    `drop-shadow` を複数重ね、それぞれに異なる色（赤と青）を指定することで、アナログモニタが故障した際のような色のズレを擬似的に再現しています。
*   **HTML5 Canvas Noise**:
    `ImageData` を直接操作してピクセルごとにランダムなグレースケール値を生成することで、オーバーレイとしての砂嵐（ノイズ）を作成しています。

### 2. 聴覚演出 (Audio Implementation)
*   **Web Speech API (SpeechSynthesis)**: 
    ブラウザ標準の音声合成エンジンを使用。`pitch`（ピッチ）を極限まで下げ、`rate`（速度）を遅くすることで、人間離れした不気味な囁き声を生成しています。
*   **Web Audio API**: 
    `OscillatorNode`（発振器）と `GainNode`（音量制御）を使い、クリック操作に合わせて「ソー（のこぎり波）」などの波形を短時間鳴らすことで、デジタル的なグリッチ音を動的に合成しています。

### 3. ブラウザAPIの活用 (Browser APIs)
*   **Page Visibility API**: 
    ユーザーが別のタブに移動したことを `visibilitychange` イベントで検知し、ドキュメントのタイトルを「逃げられない」等に書き換えることで、ブラウザの外側にまで恐怖を拡張しています。
*   **Composition & Clipping**:
    `clip-path` プロパティを使用して、外部画像を使わずにCSSだけでWin95風のファイル・フォルダアイコンを描画しています。

### 4. アニメーションとインタラクション (Interactions)
*   **Window Management**:
    DOM操作によるウィンドウの生成・ドラッグ・重ね順（z-index）の制御を行っており、`requestAnimationFrame` を介さない軽量な実装でレトロな挙動を再現しています。
*   **CSS Masking/Pseudo-elements**:
    `::before` や `::after` を活用して、既存のHTML構造を汚さずに「反射する顔」や「滲み出る血」といったレイヤーを追加しています。

## 開発者向けノート
*   **画像レス実装**: 
    ほとんどの視覚効果はCSS、SVGデータURI、およびCanvasで完結しており、外部アセットの読み込み待ちなしで即座に演出が開始されます。
*   **非同期タイピング**: 
    `setTimeout` による再帰呼び出しを用い、人間が打っているような（または何かが憑依して打っているような）不自然な速度のタイピング演出を制御しています。
